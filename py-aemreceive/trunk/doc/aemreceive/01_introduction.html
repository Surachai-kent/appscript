1. Introduction

About this tutorial

This tutorial shows how to construct a simple scriptable faceless background application, MiniTC, using the aemreceive package and its sfba sub-module. This tutorial assumes some basic knowledge of the Python language, the distutils and py2app extensions to it, and sdef resources.

About MiniTC.app

MiniTC is a cut-down version of the TextCommands application that provides AppleScript users with powerful text processing facilities. Most of TextCommands' event handlers are implemented as simple Python functions that wrap Python's built-in and standard library text tools in just a few lines of code. To demonstrate aemreceive's main features, MiniTC implements three of these commands: unicode numbers, unicode characters and strip.

Requirements

In addition to aemreceive and its dependencies (see aemreceive's README.txt for details), you will also require py2app to build the MiniTC.app application bundle.

Notes

  • The FBA_tutorial project already includes the MiniTC.icns, MiniTC.sdef and setup.py files needed to build MiniTC.app. These files may also be used as templates when creating new scriptable FBA projects.
  • The supplied setup.py file hardcodes the paths to sdp and Rez (used to generate the MiniTC.rsrc file from MiniTC.sdef); you may need to modify one or both of these before running setup.py depending on where sdp and Rez are located on your system.
  • The supplied MiniTC.sdef uses the older, pre-10.4, format. OS 10.4+ users may convert this file to the new format if they wish; see the sdef manpage for details.
