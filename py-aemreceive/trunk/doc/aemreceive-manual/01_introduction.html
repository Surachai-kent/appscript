1. Introduction

What is aemreceive?

aemreceive is simple Apple event handling framework for Python-based applications. A simple procedural API allows Python functions to be installed as Apple event handlers, and provides convenient automatic packing and unpacking of event parameters and return values/errors.

Note that aemreceive does not provide built-in Apple Event Object Model support; applications that need to work with Apple event object specifiers ('references') may be better served using PyObjC and Mac OS X's own Cocoa Scripting framework.

The aemreceive package also includes a convenience module, sfba, that makes it quick and easy to construct scriptable faceless background applications to act as AppleScript language 'extensions', similar to Mac OS X's own Keychain Scripting and URL Access Scripting tools.

This documentation assumes some basic knowledge of the Apple Event Manager API and how to use it.
