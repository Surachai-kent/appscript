About appscript

Python appscript (py-appscript) is a high-level, user-friendly Apple event bridge that allows you to control scriptable Mac OS X applications using ordinary Python scripts. Appscript makes Python a serious alternative to Apple's own AppleScript language for automating your Mac.

For example, to get the value of the first paragraph of the topmost document in TextEdit:

app('TextEdit').documents['Read Me'].paragraphs[1].get()

This is equivalent to the AppleScript statement:

tell application "TextEdit"
    get paragraph 1 of document "Read Me"
end tell

Appscript provides:

  1. an Apple event based RPC mechanism for sending commands to applications and receiving their results
  2. a simple embedded query language for constructing references to an application's object model
  3. the automatic translation of these references into human-readable form using application-defined terminology
  4. a powerful interactive help system for exploring applications' dictionaries and object models
  5. a clean, object oriented-like syntax for ease of use.
