The findapp module

The findapp module is used to obtain the full path to an application given it file name, bundle ID, or creator code. It exports the following three methods:

byname(name) -- Find the application with the given name and return its full path. 
    name : str -- application's name, e.g. 'Finder.app'. The '.app' suffix is
            optional. Absolute paths are also accepted.
    Result : str

byid(id) -- Find the application with the given bundle id and return its full path.
    id : str -- bundle id, e.g. 'com.apple.textedit'
    Result : str

bycreator(creator) -- Find the application with the given creator type and return
            its full path.
    creator : str -- four-char code, e.g. 'ttxt'
    Result : str
