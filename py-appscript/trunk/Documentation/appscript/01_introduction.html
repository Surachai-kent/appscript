What is appscript?

Appscript is a high-level, user-friendly MacPython-to-Apple Event Manager bridge that allows you to control scriptable OS X applications using ordinary Python scripts. Install appscript and MacPython becomes a serious alternative to Apple's own AppleScript language for automating your Mac.

For example, to get the value of the first paragraph of the topmost document in TextEdit:

app('TextEdit').documents[1].paragraphs[1].get()

This is equivalent to the AppleScript statement:

tell application "TextEdit"
    get paragraph 1 of document 1
end tell

Appscript builds upon lower-level Python packages (aem, osaterminology) to provide:

  1. a high-level RPC mechanism for sending commands to applications via Apple events
  2. a mechanism for converting data between common Python and Apple event types
  3. a simple embedded query language for identifying one or more objects in an application's object model
  4. a mechanism for representing these object model "references" in human-readable form based on application-defined terminology
  5. a powerful integrated help system for interactively exploring application terminologies and object models
  6. a clean, object oriented-like syntax for ease of use.
